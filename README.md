# NLP Sentiment Analysis using Watson BERT

## Project Overview

This project is a Flask-based web application developed as part of the IBM Skills Network Final Project. It uses the IBM Watson Embedded AI BERT Sentiment Analysis model to analyze user-provided text and classify its sentiment.

## Features

- Sentiment analysis using IBM Watson NLP BERT model
- Returns sentiment label and confidence score
- Flask web interface
- Error handling for invalid input
- Python package implementation
- Unit testing with unittest
- Static code analysis with pylint (10.00/10)

## Technologies Used

- Python 3.11
- Flask
- Requests
- IBM Watson Embedded AI Libraries
- Pylint
